# Brief Cho NotebookLM — So Sánh E-Brochure vs PDF

## 1) Mục tiêu bài trình bày

| Hạng mục | Nội dung |
|---|---|
| Mục tiêu chính | Chứng minh E-brochure mang lại hiệu quả kinh doanh và vận hành tốt hơn PDF tĩnh |
| Bối cảnh | Team hiện có thể gửi PDF cho khách, nhưng cần kênh tương tác + đo lường + cập nhật nhanh |
| Kết quả mong muốn | Stakeholder đồng thuận dùng E-brochure làm định dạng chính cho campaign |

## 2) Đối tượng người nghe

| Nhóm | Mối quan tâm chính |
|---|---|
| Marketing | Tốc độ ra chiến dịch, linh hoạt thay nội dung |
| Sales/CRM | Tăng click mua, tăng lead chất lượng |
| Brand/Creative | Trải nghiệm thương hiệu, visual đồng nhất |
| Management | Hiệu quả đầu tư, dữ liệu để ra quyết định |
| Operations | Giảm thao tác thủ công, giảm lỗi version |

## 3) Bảng so sánh cốt lõi

| Tiêu chí | E-brochure tương tác | PDF tĩnh | Ý nghĩa kinh doanh |
|---|---|---|---|
| Trải nghiệm người xem | Có hotspot, popup sản phẩm, gallery ảnh, video, CTA mua ngay | Chỉ xem nội dung tĩnh | E-brochure rút ngắn hành trình từ xem sang hành động |
| Khả năng chuyển đổi | Click trực tiếp tới trang sản phẩm, form lead tích hợp | Người dùng phải tự tìm link hoặc liên hệ thủ công | Tăng cơ hội chuyển đổi và giảm rơi rụng |
| Cập nhật nội dung | Chỉnh trực tiếp trong admin, publish gần realtime | Mỗi lần sửa phải xuất lại file và gửi lại | Giảm thời gian vận hành, phản ứng nhanh theo campaign |
| Quản trị media | Media Library, tái sử dụng ảnh/video, sync Cloudinary, xóa chủ động | Dễ phát sinh nhiều bản file rời rạc | Giảm trùng lặp dữ liệu và sai sót khi dùng asset |
| Đo lường | Có analytics event (view/click/lead/video), chuẩn hóa event schema | Gần như không có dữ liệu hành vi | Ra quyết định dựa trên dữ liệu thay vì cảm tính |
| Cá nhân hóa theo campaign | Có branch/tab theo bộ sưu tập hoặc thông điệp | Thường 1 file cho nhiều mục tiêu | Thông điệp rõ hơn theo từng nhóm sản phẩm |
| Khả năng mở rộng | Mở rộng block, hotspot, API import sản phẩm PNJ | Mở rộng khó, phụ thuộc thiết kế lại | Hỗ trợ tăng trưởng dài hạn |
| Kiểm soát version | 1 link live, nội dung luôn mới nhất | Nhiều file PDF khác version | Giảm rủi ro gửi nhầm bản cũ |
| Tính liên kết hệ thống | Có thể tích hợp Supabase/Cloudinary/API import | Khó tích hợp luồng dữ liệu | Tạo nền tảng cho tự động hóa marketing/sales |

## 4) Điểm “hay” nổi bật với khách hàng

| Điểm nổi bật | Mô tả ngắn | Giá trị với khách hàng |
|---|---|---|
| Chạm để xem sản phẩm | Hotspot mở popup ngay tại vị trí quan tâm | Nhanh, trực quan, không bị gián đoạn |
| Xem nhiều góc ảnh/video | Gallery ảnh + video ngay trong flow xem | Tăng độ tin tưởng trước khi mua |
| Mua ngay không vòng vo | CTA dẫn thẳng trang sản phẩm | Giảm số bước, tăng khả năng chốt |
| Trải nghiệm mobile-first | Tối ưu cho hành vi xem trên điện thoại | Dễ tiếp cận nhóm người dùng chính |

## 5) Ưu điểm cho stakeholder nội bộ

| Stakeholder | Lợi ích trực tiếp |
|---|---|
| Marketing Lead | A/B nội dung nhanh, thay banner/visual theo hiệu quả thực tế |
| Sales Manager | Theo dõi sản phẩm nào được click nhiều để ưu tiên tư vấn |
| Brand Manager | Giữ chuẩn visual và font thương hiệu trên 1 hệ thống nhất quán |
| Operation/Admin | Chủ động cập nhật không cần chờ vòng thiết kế lại PDF |
| Management | Có dữ liệu để đánh giá ROI chiến dịch |

## 6) Phân tích chi phí cơ hội (Opportunity Cost)

| Nếu tiếp tục dùng PDF là chính | Nếu chuyển sang E-brochure làm chính |
|---|---|
| Chậm vòng cập nhật khi có thay đổi sản phẩm/giá/asset | Cập nhật nhanh, giảm time-to-market |
| Thiếu dữ liệu hành vi để tối ưu | Có dữ liệu click/view/lead để tối ưu theo tuần |
| Dễ phân mảnh version khi gửi nhiều nhóm | Một nguồn nội dung live duy nhất |
| Khó mở rộng tích hợp số hóa | Sẵn nền tảng để mở rộng automation |

## 7) Đề xuất thông điệp chốt cho presentation

| Slide chốt | Nội dung đề xuất |
|---|---|
| Key Message 1 | E-brochure không thay PDF đơn thuần, mà nâng cấp thành kênh chuyển đổi có dữ liệu |
| Key Message 2 | Giá trị lớn nhất: nhanh hơn trong vận hành + tốt hơn trong trải nghiệm + rõ hơn trong đo lường |
| Key Message 3 | Nên dùng PDF như tài liệu phụ, E-brochure là kênh chính cho campaign số |
